# 🏠 Room Analyzer - ระบบวิเคราะห์และให้คะแนนห้องพัก (v2.0 Serverless)

เครื่องมือช่วยตัดสินใจเลือกที่พักอาศัย โดยใช้หลักการวิเคราะห์ข้อมูลและให้คะแนนตามน้ำหนักความสำคัญที่คุณกำหนดเอง (Weighted Scoring System) คำนวณผลลัพธ์ทันทีบนเบราว์เซอร์ของคุณ

## ✨ คุณสมบัติเด่น
* **Zero Server:** รันผ่านไฟล์ Static (HTML/JS) 100% ไม่ต้องติดตั้ง Python หรือ Database
* **Custom Weights:** กำหนดค่าน้ำหนักคะแนนตามไลฟ์สไตล์ของคุณเอง
* **Monthly Cost Estimator:** คำนวณค่าเช่ารวมค่าน้ำ ค่าไฟ และค่าซักผ้ารายเดือนจริง
* **Comparison Dashboard:** จัดอันดับห้องพักที่คุ้มค่าที่สุดให้อัตโนมัติ
* **Data Portability:** นำเข้าและส่งออกข้อมูลห้องพักผ่านรูปแบบ JSON ได้ง่ายๆ

## 🚀 วิธีการใช้งาน
1.  **เปิดใช้งาน:** เพียงเปิดไฟล์ `index.html` ผ่าน Browser หรือเข้าผ่านลิงก์ GitHub Pages
2.  **ตั้งค่า (Settings):** กด "แสดง/ซ่อน" เพื่อกำหนดงบประมาณและน้ำหนักความสำคัญ (0-10)
3.  **เพิ่มข้อมูล:** กด "เพิ่มห้องพักใหม่" และกรอกข้อมูลจากประกาศเช่าหรือที่ไปดูหน้างานจริง
4.  **ดูผลลัพธ์:** ระบบจะคำนวณคะแนน (เต็ม 10) และจัดอันดับให้ทันทีด้านล่างสุด
